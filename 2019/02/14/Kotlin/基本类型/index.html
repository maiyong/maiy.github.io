基本类型
Post：2019-02-14 16:12:22 Tags：/ Kotlin / Visit：

参考链接

Kotlin 提供的基本类型与java类似，一共有Double、Float、Long、Int、Short、Byte、Char、Boolean、String等，不同的是，Char不属于数字类型

数字类型

Kotlin 不允许较小类型隐式转换为较大类型

Type Bit width
Double 64
Float 32
Long 64
Int 32
Short 16
Byte 8

字符

不可以和数字类型直接比较，需要转换为数字类型

无符号整形

无符号整形从Kotlin1.3开始可用，kotlin.UByte … kotlin.ULong，长度是2的Bit次方
